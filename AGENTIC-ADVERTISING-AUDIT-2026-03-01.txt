   digital advertising ecosystem, evaluating how well each site supports
   interaction by AI agents — automated systems that browse, read, fill
   forms, and extract structured information on behalf of users.

   exposes the browser's accessibility tree (the same structure used by
   screen readers and automation tools). No visual rendering was used.

   Vercel is the benchmark every site in this cohort should measure against.
   Score: 82/100 — Grade B. No site in the cohort achieved an A grade;
   Vercel comes closest and defines what "AI agent ready" looks like today.

   What Vercel gets right:
   ✓ /llms.txt — comprehensive, spec-compliant, natively authored. Not
     auto-generated by a plugin — deliberately crafted for AI consumption.
   ✓ Perfect interactive element coverage — every button, link, and input
     carries a meaningful accessible name.
   ✓ Complete Organization JSON-LD — name, url, logo, sameAs all populated.
   ✓ Full Open Graph coverage — correct og:type, og:title (not "Home"),
     og:image, og:description, og:locale all present.
   ✓ Zero bot resistance — fully accessible headlessly with no CAPTCHA,
     no networkidle issues, no anti-bot barriers.
   ✓ Flawless agent navigation journey — every step from landing to CTA
     to form to return completes without friction.

   What would push Vercel to an A:
   • Add /agents.md — an emerging convention for documenting what actions
     agents are explicitly permitted to perform on the site.
   • Heading hierarchy fix — the homepage uses h2 → h4 directly (skipping
     h3), which violates semantic heading order.
   • Expand llms.txt — the current file is good; adding API documentation
     and changelog sections would make it exceptional.

   The takeaway: Vercel's /llms.txt is not generated by a plugin — it
   was deliberately authored. That intentionality is the defining difference
   between Vercel and the rest of the cohort.

6. COHORT-WIDE FINDINGS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   6a. The og:title "Home" Bug (5 domains affected)
   ─────────────────────────────────────────────────
   magnite.com, integralads.com, openx.com, taboola.com, and criteo.com
   all set og:title to "Home" — a Yoast/WordPress misconfiguration where
   the homepage title override isn't applied in Open Graph settings. Social
   shares and AI content summaries receive the non-descriptive title "Home".
   Fix: set the SEO title in Yoast's homepage settings. One-field change.

   6b. Best Social Link Labeling — PubMatic
   ─────────────────────────────────────────
   PubMatic uses "PubMatic on Facebook", "PubMatic on X (Twitter)",
   "PubMatic on LinkedIn" — explicit brand + platform labels. This is the
   pattern every site should adopt. Most of the cohort uses icon-only links
   or abbreviations ("LI", "IG", "FB") that agents cannot interpret.

   6c. Best Navigation Labeling — OpenX
   ─────────────────────────────────────
   OpenX content cards carry full multi-sentence descriptions in link text,
   enabling agents to understand product value propositions without clicking.
   This is the most agent-friendly content pattern in the cohort.

   6d. Most Complete Lead-Gen Form — StackAdapt
   ─────────────────────────────────────────────
   StackAdapt's demo request form has 9 fully labeled fields (First Name,
   Last Name, Business Email, Phone, Company, Job Title, Company Country,
   Annual Budget, Work Type). Best-in-class form accessibility for
   agent-driven lead capture.

   6e. Best Skip Link Implementation — Taboola
   ─────────────────────────────────────────────
   Taboola is the only site with both "Skip to Main Content" and "Skip to
   Footer" links. Most sites in the cohort have none.

   6f. Biggest Structured Data Gap — LiveRamp
   ───────────────────────────────────────────
   LiveRamp (built on Webflow, not WordPress) has zero WebPage/WebSite/
   Organization schema — only Wistia VideoObject embeds. For a major data
   company with "Agentic AI Orchestration" in their product nav, having no
   machine-readable entity declaration is the largest structured data gap
   in the cohort.

   6g. The Self-Block — HUMAN Security
   ────────────────────────────────────
   humansecurity.com is completely blocked by their own BotGuard product
   with a "Press & Hold to confirm you are a human" challenge. The company
   selling bot protection has applied it to their own homepage. Score: 3/100.

      Applicable to: virtually every site in the cohort.

      It reflects editorial decisions about what content matters for AI.
      Every adtech company should own this file intentionally, not as an
      afterthought from a plugin update.

      summarized. Vercel's audit revealed this as the next emerging standard
